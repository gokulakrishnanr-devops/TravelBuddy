# TravelBuddy
MERN-based travel booking platform with JWT auth, hotel booking, MongoDB, Docker support.
TravelBuddy is a full-stack MERN travel booking platform developed during the Sure Trust DevOps Internship under the guidance of team leader Arya Prakash Singh. The project is built using MongoDB, Express.js, React.js, and Node.js, with Docker-based containerization for deployment and environment management.

The platform provides a centralized travel management system where users can explore destinations, search and book hotels, reserve transportation, manage travel packages, save wishlist items, and discover regional food recommendations. The application follows RESTful API architecture with a modular backend structure for scalability and maintainability.

The backend is developed using Express.js and MongoDB with Mongoose ODM for schema modeling, validation, relationships, and dynamic database queries. Secure JWT authentication and bcrypt password hashing are implemented to protect user accounts and private routes.

Key features include hotel listing and booking management, package reservation, destination management, transportation services, profile management, wishlist functionality, and MongoDB-based filtering and search operations using operators such as $or, $gte, $lte, regular expressions, sorting, and population.

Docker and Docker Compose are integrated to containerize both frontend and backend services, ensuring consistent deployment environments and simplified setup across systems.

This project demonstrates practical implementation of full-stack development, backend engineering, REST APIs, MongoDB document modeling, authentication systems, Docker containerization, CRUD operations, and scalable application architecture inspired by platforms such as Airbnb, Booking.com, and MakeMyTrip.
